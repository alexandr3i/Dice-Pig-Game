# Pig Game (Dice)

A simple 2-player dice game (Pig) built with **HTML / CSS / JavaScript**.

## Rules
- Click **Roll dice** to add the dice value to the *current score*.
- If you roll a **1**, you lose the current score and the turn switches to the other player.
- Click **Hold** to add the current score to your total score.
- First player to reach **100** wins.
- **New game** resets everything.

## Tech
- JavaScript
- No frameworks

## Run locally
Just open `index.html` in your browser or follow the link in the about section.
